## Context

CLAUDE.md 已有「VM 部署自檢要求」規則（外部 IP 驗證），這條規則在其下方補充 API 服務的加強版要求，不取代原有規則。

## Goals / Non-Goals

**Goals:**
- 在「VM 部署自檢要求」下方插入測試矩陣規則段落
- 規則包含矩陣格式範例，讓 Claude 知道要列哪些欄位

**Non-Goals:**
- 不修改現有的外部 IP 驗證規則
- 不針對非 API 服務（純靜態網站）要求矩陣

## Decisions

規則採「先列矩陣、再逐一驗」兩步驟結構，矩陣欄位固定為：端點、Method、驗證情境。
在 CLAUDE.md 內嵌一個簡短的矩陣格式範例，讓規則自帶示意，不需要另開文件。

## Risks / Trade-offs

[風險] 矩陣步驟增加部署流程的摩擦 → 接受，只對 REST API 服務觸發，純靜態網站不受影響。
