## Why

數位學伴（大學伴）在線上一對一陪讀時，缺乏現成的互動式英語教材工具，只能依賴靜態 PDF 或螢幕共享課本，互動性低、學生容易分心。本網站提供對應台灣教育部課綱的四年級英語自製教材，讓大學伴能帶著學生逐單元學習，結合單字、句型、對話與小測驗，提升線上陪讀的效率與趣味性。

## What Changes

- 建立靜態教材網站，涵蓋四年級英語全年度課程（依課綱主題 8 單元）
- 提供單字卡片（含圖示與注音輔助）、句型練習、對話情境閱讀
- 提供每單元隨堂小測驗，讓大學伴即時掌握學生學習狀況
- 提供大學伴「陪讀提示」面板，每單元附教學引導語與互動問題範例
- 所有內容均為自製，依台灣教育部英語課綱設計，不涉及翰林版權

## Capabilities

### New Capabilities

- `unit-navigator`: 課程首頁，顯示 8 單元清單與各單元主題，作為學習入口
- `vocabulary-cards`: 每單元單字卡，含圖示、英文、中文、注音，支援逐張翻頁
- `sentence-patterns`: 每單元核心句型展示與替換練習（填空互動）
- `dialogue-reader`: 每單元對話情境閱讀，角色分色，大學伴與學生可分角色朗讀
- `unit-quiz`: 每單元結尾小測驗（選擇題 5 題），即時顯示得分
- `tutor-guide`: 大學伴面板，每單元附教學重點提示、常見錯誤提醒、互動引導語

（無現有 spec，全為新建）

## Impact

- 純前端靜態網站（HTML/CSS/JavaScript），無需後端，可部署至 GitHub Pages
- 無外部 API 依賴，教材內容硬編碼於 JS 資料檔
- 目標裝置：桌機（大學伴端）為主，需考慮平板（學生端）相容性
