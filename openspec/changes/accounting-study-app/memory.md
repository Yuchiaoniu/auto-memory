## 重要公式與概念

### 銷貨成本公式（定期盤存制）
```
期初存貨 + 進貨 = 可供銷售商品成本
可供銷售商品成本 - 期末存貨 = 銷貨成本
```

### 損益表結構
```
銷貨收入
減：銷貨退回與折讓
減：銷貨折扣
────────────────
銷貨淨額
減：銷貨成本
────────────────
銷貨毛利
```

### 付款條件 1/10, n/30
- 10天內付款享1%折扣
- 最遲30天內付清全額
- 超過10天失去折扣，付全額

### FOB 條款
- FOB shipping point：貨離起運點所有權移轉買方，運費買方出
- FOB destination：貨到目的地才移轉，運費賣方出

### 四種調整類型
| 類型 | 調整前科目 | 調整分錄 |
|------|----------|---------|
| 預付費用 | 資產（預付XX）| 借費用/貸資產 |
| 預收收入 | 負債（預收XX）| 借負債/貸收入 |
| 應計費用 | 無入帳 | 借費用/貸應付XX |
| 應計收入 | 無入帳 | 借應收XX/貸收入 |

### 會計循環八步驟
```
①分析交易→②日記帳→③分類帳→④調整前試算表
→⑤調整分錄→⑥調整後試算表→⑦財務報表
→⑧結帳分錄→結帳後試算表
```

### 每期文件清單
- 三份試算表：調整前、調整後、結帳後
- 四份財務報表：綜合損益表、權益變動表、資產負債表、現金流量表
- 一份分類帳（整年持續更新）
- 三種分錄（同一本日記帳）：平日、調整、結帳
