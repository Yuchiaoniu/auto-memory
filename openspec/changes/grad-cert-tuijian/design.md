## Context

大三下（2026年5月），推甄時間點為10～12月，距今約5個月。目標學校：政大、台師大、交大資管所。核心使命：用資訊技術與數據支持教育介入，打破創傷與暴力循環。需要在有限時間內完成2張證照並整合進備審敘事。

## Goals / Non-Goals

**Goals:**
- 6月底前通過 AWS CLF-C02
- 8月底前完成 Google Data Analytics（Coursera）
- 9月完成備審資料，敘事清晰、與使命一致
- 10～12月完成三所推甄申請

**Non-Goals:**
- 不追求第三張證照（ITIL、PL-300 暫不列入）
- 不在備考期間分心過多其他學習項目

## Decisions

- **選 CLF-C02 而非 AZ-900**：AWS 市占最高、生態最中立，對交大（技術）與政大（管理）都說得通
- **選 Google Data Analytics 而非 PL-300**：涵蓋 SQL、R、Tableau，範圍更廣；Google 品牌在學術/研究圈更中立；與「用數據衡量教育影響力」的研究方向直接對應
- **備審敘事先行**：9月撰寫時以「使命→技術能力→研究方向」為軸，讓證照成為佐證而非主體

## Risks / Trade-offs

- [大三下課業壓力] → 優先保住學期成績，CLF-C02 備考壓縮在週末與零碎時間
- [Coursera 認證完成時間較長] → 7月初立即開始，不拖到8月才啟動
- [三所定位不同] → 備審資料需針對各所微調研究計畫書，台師大強調教育科技，交大強調技術深度，政大強調管理視角
