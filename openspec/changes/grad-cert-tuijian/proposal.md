## Why

大三下資管生，計畫於 10～12 月推甄政大、台師大、交大資管所。需要在推甄前取得 1~2 張有說服力的證照，並圍繞「用資訊技術支持教育介入、打破創傷與暴力循環」的核心使命，建立一條清晰的備審敘事軸線。

## What Changes

- 完成 AWS Cloud Practitioner（CLF-C02）認證
- 完成 Google Data Analytics（Coursera）認證
- 撰寫備審資料，以上述兩張證照支撐核心使命敘事
- 提交三所推甄申請

## Capabilities

### New Capabilities

- `aws-clf-c02`: 備考與通過 AWS Cloud Practitioner 認證，建立雲端技術基礎，對應三所推甄需求
- `google-data-analytics`: 完成 Google Data Analytics Coursera 認證，建立資料分析能力，支撐教育影響力測量的研究方向
- `tuijian-narrative`: 撰寫備審資料核心敘事，將使命、技術能力、研究方向整合為一致的申請故事
- `tuijian-apply`: 完成政大、台師大、交大資管所推甄申請
