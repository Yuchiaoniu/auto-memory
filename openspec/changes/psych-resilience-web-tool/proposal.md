## Why

心理操控與情感暴力在日常環境中（校園、職場、家庭）普遍發生，但大多數人缺乏識別與應對的認知框架。現有教育體系不教授這些路徑，導致受害者、旁觀者、甚至施暴者都在無意識中重演傷害。一個開放、去病理化、以心理學路徑為核心的 Web 工具，能以最低社會成本大幅提升群體韌性。

## What Changes

- 建立互動式心理路徑地圖 Web 工具，涵蓋攻擊性路徑與正錨路徑
- 提供三種使用者視角（施暴者、受害者、旁觀者）的獨立導覽介面
- 整合微積分框架作為心理狀態變化的視覺化隱喻
- 提供情境模擬與決策樹，幫助旁觀者找到最小有效介入點
- 所有內容以中文為主，術語附英文對照，開放 GitHub 公開存取

## Capabilities

### New Capabilities

- `path-map`: 心理路徑地圖核心——攻擊性路徑與正錨路徑的完整索引，每條路徑含定義、機制、辨識訊號、反制方式
- `user-lens`: 三種使用者視角切換介面——施暴者、受害者、旁觀者各有獨立的路徑導覽與行動建議
- `calculus-visualizer`: 微積分框架視覺化——一階導數、積分、梯度、臨界點的互動式說明
- `intervention-tree`: 旁觀者決策樹——根據情境（能否說話、熟悉程度、當下狀態）給出最省力介入建議
- `pattern-interrupt`: 模式中斷指南——不出聲、不暴力的當下介入動作庫

## Impact

- 純前端 Web 應用，無需後端，可部署於 GitHub Pages
- 技術棧：HTML / CSS / JavaScript（或輕量框架如 Vue/React）
- 資料以靜態 JSON 儲存所有路徑內容
- 開放授權，鼓勵社群貢獻與翻譯
