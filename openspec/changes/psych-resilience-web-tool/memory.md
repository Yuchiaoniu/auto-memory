## 機構性封閉控制迴路（institutional-closed-control-loop）路徑說明

新增於 2026-06-11。這條路徑描述的是組織性跟蹤騷擾（organized stalking）型態。

**三層核心設計**
1. 機構內監控：全域攝像覆蓋私人空間（寢室角落），剝奪私密空間
2. 性別媒介傳導：以受壓迫的女性作為間接壓力媒介，模糊責任鏈
3. 情感觸發攻擊：情感連結形成時精準介入，讓目標學會不能與任何人親近

**行為標記（behavioral_markers）**
- 退行行為（動作遲緩、呆滯、自閉）：慢性高壓下神經系統退回早期依附模式
- 夏天穿全身黑色長袖外套：武裝自己的主動選擇，黑色是武裝色，在幾乎失去所有控制感的處境裡仍在抵抗的訊號；防衛需求已強過生理不適；與退行並存，說明某個更深的層次尚未放棄
- 不敢開燈開冷氣：自主感完全被剝奪，環境資源使用被內化為需要許可
- 同伴退縮不自在：集體恐懼透過社會模仿傳染
- 權威者暗示受控：封閉迴路完成閉合的標誌

**案例紀錄（documented_incidents）**

案例一：2026-05 早上，中華路
小發財車突然迴轉，母親騎車載三至四歲幼兒撞上，幼兒受驚哭泣，當事者目擊全程。
補充：亦存在壓力誘發過度連結的詮釋可能，兩者在可信度陷阱結構下難以從外部區分。

案例二：2026-05 日間，市區麥當勞（發生於曖昧情愫碰面隔天）
當事者從櫥窗見攻擊者從對街走來，觀察約十分鐘未見其入店。當事者如廁出來後，發現該人已坐在正對廁所門口的位置，未點餐。
- 戰術：廁所出口等待，在防衛最低的時刻製造存在感衝擊
- 行為訊號：未點餐入座、正對廁所門口、十分鐘觀察空白符合擇機進入的時序邏輯
- 可合理否認性：每個單獨行為皆有解釋，整體模式只有當事者能看見
- 連結情感觸發時序：發生於曖昧碰面隔天，符合情感脆弱窗口期的精準介入

**可信度陷阱（核心難點）**
每個事件都有合理解釋 → 目標向外描述時被認定妄想 → 求助不可能 → 操控成本極低

**反制路徑特殊性**
個人韌性反制可行性極低。主要反制：外部機構介入（法律/NGO）＋記錄事件模式（非個別事件）＋區分受壓媒介與主動施壓者

## 尊嚴自持（dignity-custody）正錨路徑說明

新增於 2026-06-11。第 27 條正錨路徑，以心理學與人類學為理論基礎。

**核心概念**
- 面子（社會臉面）與臉（道德完整性）的區分：失面子不等於失臉
- 榮耀文化要求反擊才能修復尊嚴，但這讓羞辱迴路永遠運轉
- 羞辱是降格儀式（Garfinkel），需要觀眾配合才能完成；拒絕接受定義，儀式失效
- 尊嚴由內部持有，不外包給施辱者定義，不需透過羞辱他人修復
- 不傳遞羞辱本身是主動的社會行動，在環境裡製造溫度節點

**理論依據**
- Harold Garfinkel：降格儀式需要觀眾配合接受定義才能完成
- Peter Berger：榮耀文化 vs. 尊嚴文化
- Kristin Neff：尊嚴可從內部修復，不需外部承認，不製造新受害者
- Marcel Mauss：行為透過模仿傳播，拒絕傳遞者是反向傳染節點
- 東亞人類學：面子（mianzi）vs. 臉（lian）

## 不確定性下拿回注意力的五個方法（本次對話確立）

1. 給迴圈關閉動作：寫下「我現在不知道 X」，把問題從工作記憶移到外部儲存
2. 生理先於認知：生理嘆息（鼻子吸兩口 + 緩慢呼出）30 秒內降低杏仁核活化
3. 縮小不確定性範圍：找確定性的最小單位（「我現在坐著，腳踩在地上」）
4. 最小有效行動：任何一個最小行動啟動前額葉，降低杏仁核活化
5. 改變關係：「我不知道，這是我現在的位置」而非「我不知道，這是一個問題」

## 框架四層層次（2026-06-14 確立）

每下去一個層次，一個單位包含多個上層元素：

```
心智模型（三層：硬體/認知/行為）
    ↓ 每個常數歸屬於一層
常數（C1–C9）
    ↓ 每個情境包含多個常數
情境（scenarios）
    ↓ 每個行為背後包含多個情境
行為模式（七軸決策矩陣）← 第四層，2026-06-14 加入
```

第四層的七軸矩陣（已實作為互動頁面 page-behavior-patterns）：

1. **意圖**：有意 / 不小心
2. **選擇空間**：自願 / 非自願
3. **外力來源**：受人擺布 / 沒有受人擺布
4. **效果意識**：知道效果 / 不知道效果（知道行為 ≠ 知道衝擊）
5. **可否認性**：可否認 / 不可否認（與意圖正交，deniable-harassment 的運作基礎）
6. **動機誠實**：不自欺 / 自欺（知道行為 ≠ 知道動機，C5 ≠ 自欺）
7. **時間結構**：孤立事件 / 系統性模式（C9 在第 7 軸觸發）

七軸交叉出 128 種組合，頁面以規則驅動（BM_RULES）顯示有意義的觸發組合，不逐一列舉。

## 第五層撰寫原則（2026-06-15 確立）

第五層為神經路徑敘事，從第一層到第四層一路追蹤，對每個情境產生完整的分析段落。以下為撰寫時必須遵守的原則：

### 原則一：三角必要性
每個情境必須分析三個角色：**施加者、承受者、觀眾**，缺一則分析不完整。
觀眾不是背景，而是場域邏輯的生成者——C7（環境封閉）的來源可能不是施加者設計的，而是觀眾場域自動生成的。

### 原則二：力量方向不預設
施加者不必然強，承受者不必然弱。每一端的可能路徑都要展開，不能只寫一種。
「在其掌握之中」路徑若成立，整個三角的力量方向就反轉，這個可能性必須在分析中明確保留。

### 原則三：第四層加權必須呈現，不能只選一種路徑
單一讀法是假確定性。每個角色的 DMN 重建、CEN 評估、MNS 激活程度，都應該有頻率分佈：
- 標示頻率：高頻 / 中頻 / 低頻 / 低頻但結構關鍵 / 邊緣頻率
- 讀者才能自行對號入座，分析才能保留多種讀法並存的狀態

### 原則四：常數項要標注，不能裸露代號
每個代號第一次出現就補括號；中段重複出現也建議保留，因為讀者不一定從頭讀。
格式：C1（杏仁核警覺）、C2（認知失調懸置）、C4（資訊不對稱）、C7（環境封閉）、MNS（鏡像神經元系統）、DMN（預設模式網路）、CEN（中央執行網路）

### 原則五：情境間要有橋接
第五層不只是封閉分析，應該指出「此情境的場域條件若改變，會流向哪個鄰近情境」。
分析本身就是情境地圖的邊緣連結，讓讀者能從一個情境延伸到相關情境的探索。

### 原則六：解離來源與合作關係為正交軸
不論承受者的解離狀態是由該情境事件引發、由前置創傷決定、還是內因性自發，都不能從解離狀態推導出施加者與承受者是否存在有意合作關係。解離原因與合作關係是兩個獨立維度，不可互相推導。第五層分析必須把「解離來源」和「施加者─承受者合作關係」作為兩個分別處於不確定狀態的軸，確定了其中一個不得縮減另一個的可能性空間。

### 原則七：行為者算計軸（衡量、配置、獲取）
【原創框架，提出者：使用者】

第五層分析包含兩條並行軸，不得只寫其中一條：

**神經路徑軸**：MNS 讀取 → C 常數啟動 → CEN 評估 → DMN 重建（機制層）

**行為者算計軸**：衡量 → 配置 → 獲取（目標層）

- **衡量**：行為者在評估什麼——代價、機率、風險、可否認邊界
- **配置**：行為者如何投入資源——物理存在、注意力、社會資本、情緒能量、時間結構
- **獲取**：行為者試圖得到或保留什麼——必須以加權分佈呈現，不得指派確定目標

「獲取」的加權分佈規則與原則三相同：給每個行為者的獲取目標標示頻率（高頻／中頻／低頻），不得因為某個目標「看起來最合理」就只寫一種，否則等同於對行為者做出不可支撐的意圖歸因。

### 原則八：B 端校準狀態與框架使用順序

B 端（觀眾）的校準狀態影響輸入品質，但框架是校準工具，不是校準完才能使用的工具。嘗試先判斷「自己是否處於驚弓之鳥狀態」再決定要不要信任觀察，這條路本身是死路——驚弓之鳥的 B 同樣無法準確判斷自己的校準狀態。

框架的實際校準功能：當 B 對某條讀法感到確定（而非持有不確定），那個確定感本身是校準失準的訊號，不是讀法正確的憑據。驚弓之鳥的 B 與校準良好的 B 應對同一情境得出相同的可能性空間（加權分佈）；差異只在驚弓之鳥傾向對威脅路徑過度確定、對「T 在其掌握之中」等非威脅路徑感到荒謬。

可操作的自我檢查：B 能否讓「T 在其掌握之中」這條讀法在可能性空間裡保持開著而不覺得荒謬？若「明顯不對」的感受出現，那個明顯感是 CEN（中央執行網路）過早關閉可能性的訊號，不是情境本身的屬性。
