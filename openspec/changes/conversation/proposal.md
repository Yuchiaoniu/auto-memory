## Why

自閉症者（ASD）在社交對話中常因不知道「如何開口」或「怎麼回應」而陷入沉默或焦慮。現有語言治療資源稀缺且費用高，需要一個隨時可用的低壓力輔助工具，讓使用者透過情境模擬練習實際對話技能。

## What Changes

- 新增對話情境模擬介面，讓使用者選擇情境（打招呼、請求幫忙、加入話題等）並練習開口
- 新增回應提示系統，當使用者不知道怎麼回應時，提供 2–3 個可選的語句範本
- 新增逐步引導流程，從「複誦句子」→「選擇回應」→「自由輸入」三個難度層次漸進練習
- 新增進度追蹤，記錄使用者練習過哪些情境、完成幾次

## Capabilities

### New Capabilities

- `scenario-browser`: 情境選單，涵蓋日常生活常見對話場景（打招呼、問路、購物、課堂互動等）
- `guided-practice`: 逐步引導練習流程，含三個難度層次（複誦 → 選擇 → 自由輸入）
- `response-hints`: 回應提示系統，在使用者卡關時顯示 2–3 個語句選項
- `progress-tracker`: 本機進度記錄，顯示已練習情境與完成次數

## Impact

- 純靜態前端（HTML / CSS / JavaScript），不需要後端，可部署於 GitHub Pages
- 進度資料存於 localStorage，不需要帳號或伺服器
- 無外部 API 依賴，離線可用
