## Ch06 考試重點選擇題解法精要（2026-06-14 整理）

### Q4：計算舉債資金成本，哪項因素無關？→ A（目前的利息費用）
- 負債成本應用「目前市場殖利率 YTM」，不用歷史合約利率
- 相關因素：舉債金額（決定權重）、未來稅率（影響稅盾）、債信評等（影響 YTM）
- 記憶錨點：「歷史利息費用 = 沉沒成本，與邊際融資成本無關」

### Q5：DDM 法（Gordon Model）求保留盈餘資金成本
- 公式：ks = (D1 ÷ P0) + g
  - D1 = 明年股利（題目說「明年」就直接代入，不乘 1+g）
  - P0 = 目前股價
  - g = 股利成長率
- 本題：(4.5 ÷ 90) + 10% = 5% + 10% = 15%
- 陷阱：選項 C=14.5% 是誤算股利殖利率時的常見錯誤

### Q7：WACC 反推資本結構（D/E 比率）→ C（1.00）
- 設負債占比 wd：wd×6% + (1-wd)×14% = 10%
- 解出 wd = 0.5（負債 50%、自有資金 50%）
- D/E = 0.5 ÷ 0.5 = 1.00
- 觀念：兩端（6% 和 14%）中間值（10%），必然各佔一半

### Q14：關於資金成本，何者正確？→ D（取決於籌集資金的使用方式）
- 資金成本取決於「投資用途的風險」，不取決於「資金來源」
- 錯誤選項拆解：
  - A：風險↑ → 資金成本↑（不是降低）
  - B：應用公司整體 WACC，不看專案資金來源
  - C：資本結構確實影響 WACC（最適資本結構存在）

### 五層解題框架（完整版，供需要深入拆解時參考）

---

#### L1 心智階段（Mental Stage）

任何行為主體（人或法人）從零開始，第一個動作不是取得資源，而是「看懂環境」。
三個心智階段形成閉環：

```
衡量（Measure）→ 配置（Allocate）→ 獲取（Acquire）→ 重新衡量 → ...
這個東西      →  資源往哪放   →  行動帶來報酬  →  用報酬再評估
值多少？         賺到怎麼辦        錢從哪來成本多少
```

| 心智階段 | 財務行為 | 主要章節 |
|---------|---------|---------|
| 衡量 | 評估資產/機會的真實價值 | Ch02、Ch04、Ch05、Ch13、Ch14 |
| 配置 | 決定資源去向與盈餘處理 | Ch07、Ch08、Ch10 |
| 獲取 | 融資——錢從哪來、成本多少 | Ch06、Ch09 |
| 輔助量測 | 提供三大階段所需的數字輸入 | Ch03、Ch11、Ch12 |
| 目標宣告 | 最大化股東財富（循環中心） | Ch01 |

---

#### L2 決策類型（Decision Type）

每道題必定落在三種決策之一：

| 決策類型 | 對應心智階段 | 核心問題 |
|---------|------------|---------|
| 評估決策（Valuation） | 衡量 | 這個東西值多少錢？ |
| 投資決策（Investment）/ 股利決策（Distribution） | 配置 | 資源往哪放？留還是發？ |
| 融資決策（Financing） | 獲取 | 錢從哪來？各管道成本多少？ |

注意：融資決策（WACC 計算）的結果，往往作為評估決策和投資決策的「折現率輸入」傳遞進來，三種決策在計算上彼此相連。

---

#### L3 符號類型（Symbol Type）

所有題目都在「計算」，只是符號不同：

| 符號類型 | 定義 | 範例 |
|---------|------|------|
| 數字公式 | 帶入數值，輸出具體數字 | P = D ÷ k = 6 ÷ 0.12 = 50 |
| 文字公式 | 帶入概念前提，輸出判斷結論 | EMH 成立 → 技術分析無法帶來超額報酬 |

兩者結構相同：輸入前提 → 套規則 → 輸出結論。差異只在輸入與輸出的符號類型。

---

#### L4 計算目的（Calculation Purpose）

計算題的核心動作，折現與溢酬同時決定折現率，定價是最後結果：

```
折現率 k = 無風險利率 r_f（折現）+ 風險溢酬 RP（溢酬）
                     ↓
         套進公式算出現值（定價）
```

| 目的 | 說明 | 激活時機 |
|------|------|---------|
| 折現（Discounting） | 把未來現金流摺回現在，反映時間的代價 | 所有 PV/NPV 計算 |
| 溢酬（Premium） | 不確定性要求的額外補償，超出無風險利率的部分 | 風險調整、CAPM |
| 定價（Pricing） | 折現 + 溢酬合體後，套公式得出公平價格 | Ch05 所有評價題 |

文字公式題的 L4 角色：降格為「解釋背景」（解釋為什麼這個邏輯成立），不執行計算。

---

#### L5 K-Constants（計算常數群）

k 群是「折現 + 溢酬」合體後的數值化表達，每個符號都對應一個具體的資本來源或用途：

| 符號 | 名稱 | 定義方式 |
|------|------|---------|
| r_f | 無風險利率 | 折現的基準（政府公債殖利率） |
| RP | 風險溢酬 | 超出 r_f 的補償，= β × 市場風險溢酬 |
| k | 必要報酬率（通稱）/ 市場殖利率 | r_f + RP；同一個數字的兩個視角：投資人角度稱「必要報酬率」（要求的最低報酬），市場角度稱「市場殖利率」（債券目前的殖利率）。兩者不同名稱但指同一個 k |
| ks | 普通股必要報酬率 | DDM 法：ks = D1/P0 + g；或 CAPM 法 |
| kd | 稅前負債成本 / 市場殖利率（YTM） | 投資人角度：要求負債給的報酬率；市場角度：債券目前的 YTM。同一個 k，看誰在描述 |
| kd(1-T) | 稅後負債成本 | kd × (1 − 稅率 T)，負債有稅盾 |
| kp | 特別股資金成本 | kp = D ÷ P0（無稅盾，不乘 1-T） |
| WACC | 加權平均資金成本 | = Σ（各來源成本 × 權重），配置的門檻 |
| β | 系統風險係數 | 溢酬的輸入參數，衡量相對市場的波動 |
| g | 股利成長率 | DDM 中 ks 的組成部分 |

**k 群在三個決策中的角色：**
- 獲取：計算各管道的 k 值（ks、kd、kp）→ 組成 WACC
- 衡量：WACC 或市場 k 作為折現率 → 算出定價
- 配置：WACC 作為門檻 → NPV/PI/IRR 比較的基準

---

#### L5 CF-Constants（現金流群）

CF 群是 k 群折現時需要代入的材料，分成兩類：輸入（現金流本身）和輸出（計算結果）。

| 符號 | 名稱 | 定義 | 角色 |
|------|------|------|------|
| D0 | 本期已發股利 | 今年（時間點零）實際發出的股利 | 輸入 |
| D1 | 下期預計股利 | 明年預計發出的股利 = D0 × (1+g) | 輸入 |
| g | 股利成長率 | 每年股利的固定成長比率（Gordon Model 用）| 輸入 |
| C | 息票（Coupon） | 債券每期支付的固定利息 = F × 票面利率 | 輸入 |
| F | 面額（Face Value） | 債券到期時歸還的本金 | 輸入 |
| n | 期數 | 債券到期年數，或折現的總期數 | 輸入 |
| P0 | 目前公平股價 / 債券現值 | 今天的合理價格，是定價步驟的**輸出**；PV 的一個特例（專指股票） | 輸出 |
| PV | 現值（Present Value） | 未來任何現金流折算到現在的價值；P0 是股票版的 PV | 輸出 |
| NPV | 淨現值 | PV(未來現金流) − 初始投資，配置決策的核心指標 | 輸出 |

**折現工具（PVIF / PVIFA）：**

**為什麼要分開算息票 PV 和本金 PV？**
```
息票（C）：每年都有一筆，共 n 次，每次金額相同
           → 「重複出現的固定金額」（年金）
本金（F）：只在第 n 年末出現一次
           → 「單筆、只出現一次的金額」
數學上無法用同一個公式同時處理兩者，必須拆開計算再加總。
```

| 工具 | 全名 | 公式 | 用途 |
|------|------|------|------|
| PVIF(k, n) | Present Value Interest Factor | 1 ÷ (1+k)^n | 折現**單筆**未來金額（如本金 F）|
| PVIFA(k, n) | Present Value Interest Factor of Annuity | [1 − 1÷(1+k)^n] ÷ k | 折現**每期固定金額**（如息票 C）的全部加總 |

PVIFA = 把 n 個 PVIF 加總後包裝成一個係數，避免逐期手算。

| 比較點 | PVIF | PVIFA |
|--------|------|-------|
| 處理的現金流 | 單筆（只出現一次）| 年金（每期重複相同金額）|
| 計算邏輯 | 折現一次 | 折現 n 次再加總 |
| 對應 Q9 的角色 | 折算第 5 年末的本金 100,000 | 折算第 1–5 年的息票各 12,000 |

**具體數字範例（Ch05 Q9，k=10%, n=5）：**
```
PVIF(10%, 5)  = 1 ÷ (1.10)^5 = 0.6209
PVIFA(10%, 5) = [1 − 0.6209] ÷ 0.10 = 3.7908
（PVIFA 也可理解為 PVIF(10%,1)+...+PVIF(10%,5) 的加總）
```

**對照參考（Ch05 問答 Q7，k=10%, n=3）：**
```
PVIF(10%, 3)  = 1 ÷ (1.10)^3 = 0.7513
PVIFA(10%, 3) = 2.4869
```

**債券完整定價公式：**
```
P = C × PVIFA(k, n)  +  F × PVIF(k, n)
    每年息票的現值加總      面額（本金）的現值
```

**債券方向規則（k 與 P 的反向關係）：**
```
k（折現率）在分母 → k ↓ → 分母變小 → P ↑
                     k ↑ → 分母變大 → P ↓
快速判斷：k < 票面利率 → P > 面額（溢價）；k > 票面利率 → P < 面額（折價）
```

**k 群與 CF 群的關係：**
```
CF 群（D1、g 或 C、F、n）  →  輸入公式  →  k 群折現  →  CF 群（P0 / PV）輸出
```
k 群是「用什麼利率折」，CF 群是「折什麼、折出來是多少」，兩者合作才能完成 L2 的定價步驟。

#### Ch06-S01｜特別股資金成本（市價法）

**Setup**  
已知特別股年股利（D）與目前市價（P₀）。

**決策問句**  
這筆特別股對公司的資金成本 kp 是多少？

**激活域與常數**  
- H3 稀缺域：資本有代價，特別股股利是公司必須支付的承諾成本  
- H2 不確定域：F2（承擔風險要求補償）——特別股無稅盾，比負債風險高，成本也較高  
- F5：成本取決於用途（稀缺域的稅盾是否存在，看資本性質）

**框架路徑**  
融資決策 → 計算各管道成本 → kp = D ÷ P₀  
（不乘稅盾係數：特別股股利不能抵稅，與負債的 kd(1-T) 不同）

**常見陷阱**  
誤套負債公式乘上 (1-T)，把特別股當成有稅盾的融資工具。

#### Ch06-S04｜舉債資金成本的無關因素（沉沒成本陷阱）

**Setup**  
題目列出四個因素，問哪一項「與計算舉債資金成本無關」。

**決策問句**  
下列哪項因素對負債邊際成本沒有影響？→ 答案：目前的利息費用（歷史合約利率）

**激活域與常數**  
- H1 時間域：資金成本是「未來」的邊際成本，不是「過去」已簽的合約利率  
- H3 稀缺域：每一塊新借入的資本都要按市場現況重新定價  
- F5：資金成本看「現在的市場要求」，不看來源的歷史記錄

**框架路徑**  
融資決策 → 計算 kd → 使用市場殖利率（YTM），而非合約利率  
相關因素：舉債金額（決定權重）、未來稅率（影響稅盾）、債信評等（影響 YTM）

**常見陷阱**  
以為「我現在付的利息費用」就是資金成本。正確邏輯：沉沒成本與邊際融資成本無關。

#### Ch06-S05｜DDM 法求保留盈餘資金成本

**Setup**  
已知：明年股利 D1、目前股價 P₀、股利成長率 g（Gordon Model 三要素全給）。

**決策問句**  
保留盈餘的資金成本 ks 是多少？

**激活域與常數**  
- H1 時間域：D1 是「明年」的股利，P₀ 是「現在」的股價，折現關係確立  
- H3 稀缺域：保留盈餘不是「免費資金」，有機會成本（股東要求的必要報酬率）  
- F3：資產價值 = 未來現金流現值（股價 P₀ 是所有未來股利的現值）  
- F5：保留盈餘的成本 = 股東的必要報酬，與盈餘的歷史累積方式無關

**框架路徑**  
融資決策 → DDM（Gordon Model）→ ks = (D1 ÷ P₀) + g  
本題計算：(4.5 ÷ 90) + 10% = 5% + 10% = **15%**

**常見陷阱**  
題目說「明年股利 D1 = 4.5」→ 直接代入，不再乘 (1+g)。  
誤算路徑：把 D1 當 D0 再乘一次 (1+g)，算出偏高數字。

#### Ch06-S07｜WACC 反推資本結構（D/E 比率）

**Setup**  
已知：WACC = 10%、稅後借款成本 kd(1-T) = 6%、股東必要報酬 ks = 14%。

**決策問句**  
公司的 D/E（負債÷股東權益）比率是多少？→ 答案：1.00

**激活域與常數**  
- H3 稀缺域：資本比例加總等於 1（w_d + w_e = 1），這是稀缺約束的數學表達  
- F4：WACC 是加權平均，公式結構提供反推方程式  
- F5：kd 和 ks 各自獨立，可以分開處理（兩個未知量減為一個）

**框架路徑**  
融資決策 → WACC 公式 → 代數反推  
```
WACC = w_d × kd(1-T) + (1-w_d) × ks
10%  = w_d × 6% + (1-w_d) × 14%
10%  = 6%w_d + 14% - 14%w_d
-4%  = -8%w_d  →  w_d = 0.5，w_e = 0.5
D/E  = 0.5 ÷ 0.5 = 1.00
```
觀念捷徑：10% 正好在 6% 和 14% 的中間點，兩端各佔一半，直接得出 D/E = 1。

**常見陷阱**  
以為反推需要特殊公式，其實只是一元一次方程式，H3 的「比例加總為 1」是解題關鍵。

#### Ch06-S14｜資金成本取決於使用方式（概念判斷題）

**Setup**  
公司考慮一個新計畫，資金可從「保留盈餘」或「銀行借款」取得。

**決策問句**  
關於資金成本，下列何者正確？→ 答案：取決於籌集資金的使用方式

**激活域與常數**  
- H3 稀缺域：任何資本都有機會成本，沒有「免費錢」這種資源  
- F5：資金成本取決於「投資用途的風險」，不取決於「資金來源」  
（保留盈餘 ≠ 免費，其機會成本 = ks）

**框架路徑**  
融資決策 → 概念判斷 → F5 直接套用  
錯誤選項拆解：  
- A：風險↑ → 資金成本↑（不是降低，方向搞反）  
- B：應用公司整體 WACC，不看「這筆錢從哪個口袋拿出來」  
- C：資本結構確實影響 WACC（最適資本結構存在，C 說法不正確）

**常見陷阱**  
以為「自己的盈餘留著用」就不用算成本。正確邏輯：保留盈餘不發股利，代表股東放棄了拿去自己投資的機會，這個放棄的報酬率就是 ks。

#### Ch07-S07｜NPV 計算與接受/拒絕決策

**Setup**  
初始投資 $9,700，年1現金流 $6,700，年2現金流 $4,500，折現率（WACC）= 10%。

**決策問句**  
這個計畫的 NPV 是多少？應該接受還是拒絕？→ 答案：NPV ≈ +110，接受

**激活域與常數**  
- H1 時間域：未來現金流必須折回「現在」才能和初始投資比較  
- H3 稀缺域：WACC 是機會成本門檻，唯有 NPV > 0 才創造股東財富  
- F3：資產價值 = 未來現金流的現值總和（NPV 公式的本質）  
- F4：IRR > WACC 等價於 NPV > 0，通過門檻才接受

**框架路徑**  
投資決策 → NPV 計算 → NPV > 0 接受  
```
NPV = -9,700 + 6,700/(1.1)¹ + 4,500/(1.1)²
    = -9,700 + 6,090.9 + 3,719.0
    = +109.9 ≈ +110 > 0 → 接受
```

**常見陷阱**  
把未來現金流直接加總（$6,700 + $4,500 = $11,200），忘記折現；或方向搞反，以為 NPV > 0 要拒絕。

#### Ch07-S11｜資本限制下應使用 PI 排序

**Setup**  
公司面臨資本限制（Capital Rationing），預算不足以投資所有 NPV > 0 的計畫，需要選出最有效率的方案組合。

**決策問句**  
在資本限制下，最適合使用哪種評估方法？→ 答案：獲利能力指數（PI）

**激活域與常數**  
- H3 稀缺域：資本有限，每一元資本都有機會成本，必須比較「每元貢獻多少現值」  
- F4：WACC 門檻決定哪些計畫的 NPV > 0，但稀缺迫使進一步排序  
- F3：PI = NPV / 初始投資 = 每投入 1 元創造的現值倍數

**框架路徑**  
投資決策 → 資本限制 → PI 排序（高 PI 優先投入）  
（不用 NPV 絕對值：大計畫 NPV 高但需要更多資本，PI 才能衡量效率）

**常見陷阱**  
直接選 NPV 最大的計畫，忽略資本限制——可能因此選了一個大計畫，卻浪費了用兩個小計畫能獲得更高總 NPV 的機會。

#### Ch07-S12｜互斥方案應以 NPV 最大者為準

**Setup**  
公司考慮兩個互斥方案 A 和 B，兩者都不能同時執行。

**決策問句**  
正確的決策準則是哪種？→ 答案：選 NPV 較大者（方案 C）

**激活域與常數**  
- H3 稀缺域：互斥意味只能選一，機會成本 = 放棄的另一個方案  
- H1 時間域：必須折現才能正確比較不同時間點的現金流  
- F3：NPV 直接反映「這個計畫讓股東財富增加多少元」  
- F7：財務決策目標是最大化股東財富，NPV 最大即財富增加最多

**框架路徑**  
投資決策 → 互斥方案比較 → 以 NPV 絕對值最大者為準

**常見陷阱**  
選 IRR 較高的方案——小計畫 IRR 可能很高，但 NPV 絕對值小，股東財富實際增加少；或選 PI 較高者，同樣的規模差異問題。互斥方案的正確指標只有 NPV。

#### Ch07-S17｜NPV 法三大特性概念判斷

**Setup**  
四個關於 NPV 法的敘述，考辨析三大特性是否正確。

**決策問句**  
下列何者為真？→ 答案：D（NPV 愈大，公司價值增加愈多）

**激活域與常數**  
- H3 稀缺域：NPV 代表計畫創造的超額現值，直接連結 F7（股東財富最大化）  
- H1 時間域：折現的前提是 WACC，不是 IRR  
- F3：價值相加性（NPV 可以跨計畫加總）  
- F4：NPV > 0 等價於 IRR > WACC（大於，不是等於）

**框架路徑**  
投資決策 → 概念判斷 → NPV 法三大特性逐一檢驗  

| 選項 | 內容 | 對錯 | 理由 |
|------|------|------|------|
| A | 不具價值相加性 | ✗ | NPV **有**加法性，可跨計畫加總 |
| B | 再投資假設與 IRR 相同 | ✗ | NPV 假設以 WACC 再投資，IRR 才假設以 IRR 再投資 |
| C | NPV > 0 時 IRR = 必要報酬率 | ✗ | NPV > 0 代表 IRR **大於** WACC，不是等於 |
| D | NPV 愈大，公司價值增加愈多 | ✓ | NPV 以金額直接衡量股東財富貢獻 |

**常見陷阱**  
混淆「NPV = 0 時 IRR = WACC」（盈虧平衡點）和「NPV > 0 時 IRR > WACC」這兩個條件；或以為 NPV 不能加總（實際上是 IRR 不能加總，NPV 才能）。

#### Ch05-S01｜效率市場假說（EMH）定義

**Setup**  
效率市場假說成立時，市場呈現什麼狀態？

**決策問句**  
哪個選項正確描述 EMH 的核心意義？

**解題路徑**

#### Ch05-S03｜發行何種證券破產風險最低

**Setup**  
公司考慮四種融資工具，問哪種對公司破產風險最低？答案：D 普通股

**決策問句**  
下列哪種融資工具讓公司面臨的破產風險最低？

**五層定位**  
- L1：獲取——融資工具的性質選擇
- L2：融資決策
- L3：文字公式
- L4：溢酬概念作為背景邏輯（股東承擔最高風險、無固定請求權）
- L5：無 k 群常數需要計算；概念上涉及 ks > kd > kp（風險與成本排序）

**解題路徑**  
普通股無固定還款義務 → 公司無法支付股利不構成法定違約 → 破產風險最低  
（債券/特別股有固定義務，無法支付即違約）

**常見陷阱**  
與「哪種融資成本最高」混淆——普通股成本最高（ks），但破產風險最低；兩者方向相反。

#### Ch05-S07｜特別股定價（永續年金法）

**Setup**  
特別股年股利 D = 6 元、投資人必要報酬率 k = 12%，求合理股價？答案：B 50 元

**決策問句**  
特別股的公平價格是多少？

**五層定位**  
- L1：衡量——資產定價
- L2：評估決策
- L3：數字公式
- L4：折現（永續年金）+ 溢酬（k 已含風險補償）→ 定價
- L5：kp = 12%（特別股必要報酬率）

**解題路徑**  
特別股 = 永續年金（無到期日，每期固定股利）  
P = D ÷ k = 6 ÷ 0.12 = **50 元**

**常見陷阱**  
與有到期日的債券定價混淆，誤用逐期折現加總公式（那是有限期現金流的方法）；永續年金只需一步除法。

#### Ch05-S09｜債券定價（有限期現金流）

**Setup**  
政府公債面值 $1,000、票面利率 8%、三年期、市場必要報酬率 10%，求投資人最高願意出的價格。

**決策問句**  
這張債券的公平價格（最高出價）是多少？

**五層定位**  
- L1：衡量——債券定價
- L2：評估決策
- L3：數字公式
- L4：折現（有限期，逐期折回）+ 溢酬（10% 已含）→ 定價
- L5：k = 10%（市場殖利率，也是折現率）

**解題路徑**  
```
息票 = 1,000 × 8% = 80 元（每年）
PV  = 80/(1.1)¹ + 80/(1.1)² + (80+1000)/(1.1)³
    = 72.73 + 66.12 + 811.36 ≈ 950.21 元
```
最高出價 = $950.21（出更高代表接受低於 10% 的報酬，違反 k 的定義）

**常見陷阱**  
與永續年金公式（Ch05-S07）混淆——債券有到期日，必須逐期折現加總面值；不能用 D÷k 一步算出。

#### Ch05-S11｜股票評價（DDM 法，零成長）

**Setup**  
股票每年固定股利 D = 2 元、投資人必要報酬率 ks = 10%，求股票合理價格？

**決策問句**  
零成長股票的公平價格是多少？

**五層定位**  
- L1：衡量——股票定價
- L2：評估決策
- L3：數字公式
- L4：折現（永續年金，無成長）+ 溢酬（ks 已含）→ 定價
- L5：ks = 10%（普通股必要報酬率）

**解題路徑**  
零成長 DDM（等同永續年金）：P = D ÷ ks = 2 ÷ 0.10 = **20 元**

**常見陷阱**  
與成長股 DDM 混淆：有成長時 P = D1 ÷ (ks - g)，分母需減去成長率；零成長時 g = 0，公式退化為 D ÷ ks。

#### Ch05-S21｜成長股評價（DDM 法，Gordon Model）

**Setup**  
已知明年股利 D1、目前股價 P0、股利成長率 g，或給定 D0（本期股利）需先乘 (1+g)。

**決策問句**  
成長股的合理股價（或 ks）是多少？

**五層定位**  
- L1：衡量——股票定價 / 獲取（若反求 ks 則為融資決策輸入）
- L2：評估決策（正向求 P）或融資決策（反向求 ks）
- L3：數字公式
- L4：折現（永續等比成長）+ 溢酬（ks 或市場 k）→ 定價
- L5：ks（必要報酬率）、g（股利成長率）、D1（明年股利）

**解題路徑**  
Gordon Model（成長型 DDM）：P0 = D1 ÷ (ks - g)  
反推 ks：ks = (D1 ÷ P0) + g

**常見陷阱**  
題目給 D0（本期股利）時，必須先算 D1 = D0 × (1+g) 再代入；直接用 D0 代入會低估股價。
