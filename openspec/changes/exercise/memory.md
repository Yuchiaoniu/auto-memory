# exercise — 知識地圖

## 核心命題 ／ 終極目標

**終極目標：開心地持續變強——在智慧上、體魄上、記憶力上、理解力上、溝通能力上。**

低壓力 → 高動力 → 外溢正向影響。
「享受生活」與「持續進步」不是對立的，可以互相增強。
同時，自身狀態能對周遭的人形成正向行為傳染。

## 路徑一：運動科學（MED 原則）

### 最小有效劑量（Minimum Effective Dose）

- 肌肥大不需要每天練、不需要練到精疲力竭。
- 文獻支持：每週每個肌群 **10 組以上**即可觸發超補償，前提是蛋白質足夠、睡眠充足。
- 「享受型生活」完全可以嵌進這個最低需求。

### 肌肉降解與合成時間窗

| 時間點 | 狀態 |
|--------|------|
| 訓練後 0h | MPB（肌肉蛋白分解）衝高 |
| 訓練後 24h | MPS（肌肉蛋白合成）衝高、開始超補償 |
| 訓練後 48h | MPB 與 MPS 雙雙回落至基準線 |

結論：每 48–72 小時給一次刺激即可；48 小時後不訓練也不會加速降解。

## 路徑二：認知／心理（動機系統）

### 自我決定理論（Self-Determination Theory, SDT）

三種動機燃料：
1. **內在動機**：做這件事本身就讓人有成就感或享受。
2. **認同動機**：知道這件事跟「我是誰」一致（身份認同型）。
3. **外在動機**：他人要求或避免懲罰驅動。

意志力只有在前兩種燃料不足時才需要消耗。  
「不費力的持續」的關鍵是把健身與唸書「認同化」——讓它成為「我本來就是這種人」的自然延伸。

### 相關概念對照

| 概念 | 核心意義 |
|------|----------|
| 認知負荷（Cognitive Load） | 當天已消耗的心智資源總量 |
| 流動狀態（Flow State） | 挑戰與能力恰好匹配時的高效無壓狀態 |
| 超日節律（Ultradian Rhythm） | 90–120 分鐘的認知高峰週期，可用來規劃深度工作 |

## 路徑三：社會動態（行為傳染）

### Christakis & Fowler（2007）三度分隔效應

- 幸福感、運動習慣、飲食選擇都會在社交網絡中傳播。
- 效應可達三度分隔（朋友的朋友的朋友）。
- 關鍵不是主動說服，而是**自身狀態本身就是可觀察的訊號**。

### 與 psych-resilience 專案的接口

- 正錨定路徑（positive anchoring）的機制：一個真實享受生活但持續進步的人，周遭的人觀察到後會自發模仿。
- 自身行為的「正確性」（authenticity）比說服更有效——對應 psych 專案的情境案例中的正向錨定路徑。

## 路徑四：生物酶降解（UPS 與蛋白酶機制）

### 主要降解系統

| 系統 | 觸發條件 | 關鍵分子 |
|------|----------|----------|
| UPS（泛素蛋白酶體系統） | 皮質醇升高、能量不足 | MuRF1、MAFbx（atrogin-1） |
| Calpain（鈣蛋白酶） | 運動後鈣離子升高 | μ-calpain、m-calpain |
| Autophagy（細胞自噬） | 長期禁食、極度訓練 | LC3、Beclin-1 |

### 抑制降解的生化開關

- **mTOR 活化** → 關閉 UPS，同時促進 MPS
- **胰島素 / IGF-1 升高** → 抑制 MAFbx 表達
- **避免 AMPK 過度活化** → 能量充足時 AMPK 保持低活性，不促分解

結論：低皮質醇 + 蛋白質充足 + 睡眠充足 → UPS 活性維持低水平 → MED 就足以讓合成大於分解。

## 路徑五：多巴胺與激素（動機的生化基礎）

### 多巴胺的本質

- 多巴胺不是「獎勵感」，而是**期待感**（wanting，而非 liking）——Berridge & Robinson（1998）。
- 基線多巴胺水平決定一個人願意付出多少努力去追求目標。
- **多巴胺經濟學**：人為製造高峰（過度刺激）後，基線會下降，導致做什麼都提不起勁。
- 運動本身會提升基線多巴胺（而非只製造短暫高峰）。

### 關鍵激素對照表

| 激素 | 促合成 / 提升動機 | 促分解 / 抑制動機 |
|------|-------------------|-------------------|
| 睪固酮 | MPS 上升、動機感強 | — |
| 生長激素（GH） | 睡眠中峰值分泌，促脂肪代謝與恢復 | — |
| IGF-1 | 抑制 UPS，強化 mTOR | — |
| 胰島素 | 抗分解、促葡萄糖攝取 | — |
| 皮質醇 | — | 活化 UPS↑、抑制多巴胺合成↓、降低睪固酮 |

### 最可量化的壓力指標

**皮質醇 / 睪固酮比值（C/T ratio）**：比值越高，代表身體處於淨分解狀態，動機系統同步受壓制。享受型低壓生活方式是維持低 C/T 比值的最直接手段。

## 心智模型架構

### L1 心智閉環（保留原財務模型結構）

```
衡量（Measure）→ 配置（Allocate）→ 獲取（Acquire）→ 重新衡量 → ...
今天狀態如何？  →  精力往哪放    →  行為帶來報酬  →  用結果再評估
```

### L2 三大激素主角（對應 L1 三個動作）

| L1 動作 | L2 激素 | 角色說明 |
|--------|--------|---------|
| 衡量 | 皮質醇 | 衡量現在的系統壓力成本，判斷處於合成還是分解狀態 |
| 配置 | 多巴胺 | 動力信號，基線水平決定願意把精力放在哪裡 |
| 獲取 | 睪固酮 | 訓練與恢復做對後的合成報酬，是獲取兌現的訊號 |

皮質醇是支配性變數：唯一能同時壓制多巴胺與睪固酮的激素。

### 三激素相互關係

```
皮質醇 ──抑制──▶ 多巴胺合成酶（酪胺酸羥化酶）→ 動力消失
皮質醇 ──抑制──▶ 睪固酮（孕烯醇酮偷竊效應）→ 合成報酬消失

多巴胺 ──反向調節──▶ 降低壓力感知 → 皮質醇小幅下降
睪固酮 ──反向調節──▶ 輕微抑制皮質醇（效果弱）
```

## 皮質醇上升的四種情境

### 類型一：生理壓力
- 訓練量超過 MED 太多（過度訓練）
- 睡眠不足（< 6 小時，GH 分泌被截斷）
- 熱量嚴重不足（身體進入危機模式）

### 類型二：認知／心理壓力
- 決策疲勞（同一天做太多決定）
- 持續焦慮與反覆思慮
- 感知到「失控」的情境（結果不可預期）

### 類型三：社交壓力
- 衝突、被批評、被評判
- 孤立感（社交隔離）
- 表現壓力（考試、報告前的持續緊張）

### 類型四：多巴胺高峰崩落後（最容易被忽略）
- 高刺激行為結束後（長時間遊戲、滑手機、咖啡因退效）
- D_peak 越高，崩落越深，崩落期間皮質醇輕微上升
- 這段時期會感到坐立難安、動力消失

### 享樂的兩種版本（關鍵區分）

| 高刺激享樂 | 低刺激享樂 |
|-----------|-----------|
| 遊戲、酒精、滑手機 | 好好吃飯、散步、聊天 |
| D_peak 衝高後崩落 | D_base 緩慢上升，無明顯崩落 |
| 皮質醇輕微上升，睪固酮受壓制 | 皮質醇維持低水平，睪固酮正常分泌 |

### L3（暫定）：三個維度

**維度一：宏觀 vs 微觀**（時間尺度）
- 宏觀：長期適應（肌肉數週生長、神經迴路數月強化）
- 微觀：單次訓練或當下瞬間的刺激強度感受

**維度二：太輕鬆 vs 太痛苦**（挑戰水平）

```
                太輕鬆              最適挑戰（心流）    太痛苦
宏觀（週/月）  長期習慣鬆弛        可持續進步          過度訓練/燃盡
微觀（日/次）  當下無聊→遊戲衝動   心流狀態            當下痛苦→逃避
```

- 太輕鬆：DMN 活躍、D_base 下降、大腦掃描下一個 D_peak
- 太痛苦：皮質醇升高、TPN 超載、逃避行為出現
- 最適挑戰：CEN 穩定在線、多巴胺維持中等水平、心流

心流三個觸發條件（Csikszentmihalyi）：清楚目標 ＋ 即時回饋 ＋ 挑戰剛好在技能邊緣。

**維度三：自主 vs 被動**（動機來源）

| | 自主端 | 被動端 |
|--|--------|--------|
| 動機類型（SDT）| 內在動機 / 認同動機 | 外在動機 / 內攝調節 |
| 完成後睪固酮 | 上升（勝利效應觸發）| 不上升或效果微弱 |
| 完成後 D_base | 補充 | 無補充或下降 |
| 皮質醇 | 維持低水平 | 持平或上升 |
| 長期效應 | 動力累積，可持續 | 動力耗盡，情緒爆發風險 |

三個維度同時成立時動力最穩定：
微觀層面 ＋ 最適挑戰 ＋ 自主 = 心流狀態，且完成後睪固酮上升，D_base 獲得補充。

**維度四：相對性配對**（雙力動態，淨值決定方向）

| 配對 | 正向力 | 負向力 | 淨值決定 |
|------|--------|--------|---------|
| 肌肉層 | MPS（肌肉蛋白合成）| MPB（肌肉蛋白分解）| MPS > MPB → 肌肉生長 |
| 多巴胺層 | D_base（基線，地板）| D_peak 後的 D_decay（崩落）| D_base 高 → 維持山腰；D_peak 高 → 短暫衝高後墜谷 |

說明：
- **MPS vs MPB**：訓練刺激、蛋白質攝取、睡眠提升 MPS；皮質醇、過度訓練、熱量不足提升 MPB。淨值為正才有肌肉成長。
- **D_base vs D_peak**：D_base 靠睡眠、運動達成、小目標達成（睪固酮介導）緩慢抬高；D_peak 靠高刺激行為（遊戲、咖啡因）急升，但 D_peak 越高崩落越深，D_base 反而被壓低。維持山腰的策略是拉高 D_base，而非追求 D_peak。

## L4：六個確定性維度（光譜格式，與 L3 結構相同）

不確定性稅機制：左端累積 → 皮質醇微升 → PFC 威脅敏感模式 → 成本估算被放大 → 迴避反應。
每個維度的左端是迴避觸發端，右端是行動可被啟動的狀態。

| 維度 | 迴避端（左）| 行動端（右）| 主要觸發 | 解決方式 |
|------|-----------|-----------|---------|---------|
| 社交安全感 | 社交威脅高 | 社交安全 | 皮質醇↑ | 只能接受，降低敏感度 |
| 自我效能感 | 能力懷疑 | 確信能做到 | 皮質醇↑ | 靠「開始」塌縮 |
| 路徑清晰度 | 路徑模糊 | 路徑清晰 | 皮質醇↑ | 靠「只做第一步」塌縮 |
| 長期價值感 | 價值不明 | 長期價值確認 | 多巴胺↓ | 靠「重新框架目標」轉化 |
| 時間可預估性 | 時間未知 | 時間可預估 | 多巴胺↓ | 靠「開始」塌縮 |
| 身份一致性 | 身份矛盾 | 身份一致 | 睪固酮缺位 | 靠「找到認同詮釋」轉化 |

### 自我應驗循環

迴避 → 任務停在未知 → 不確定性從未解決 → 下次更快觸發 → 「果然很麻煩」得到驗證 → 迴避模式強化。

打破方式：只定義第一步，限制 PFC 的模擬範圍，讓可以被塌縮的維度先塌縮，減少同時啟動的不確定性數量。

### 情境套用：筆記繳交任務

| 不確定性類型 | 在筆記任務裡的具體形態 | 可不可以解決 |
|------------|---------------------|------------|
| 社交 | 「老師/同學會不會覺得我在裝認真？」 | 只能接受，無法消除 |
| 能力 | 「我能整理出夠好的版本嗎？」 | 靠開始塌縮 |
| 執行路徑 | 「要整理哪些？格式怎麼定？」 | 靠第一步塌縮 |
| 長期價值 | 「交這份筆記對我有什麼用？」 | 靠重新框架轉化 |
| 時間成本 | 「這要花多久才能整理完？」 | 靠開始塌縮 |
| 身份認同 | 「我是在被動應付，還是在做自己的事？」 | 靠重新框架轉化 |

六種同時存在，但只有社交是真的無解——其他五種都有對應的處理手段。

## 神經與肌肉的強化機制對照

### 肌肥大：接近力竭是關鍵，不是次數

- 小重量多次數 ≈ 大重量少次數（Schoenfeld 2017），前提是都訓練到接近力竭（RIR 0–3）
- 觸發機制重要性排序：機械張力 ＞ 代謝壓力 ＞ 肌肉損傷
- 「次數多」本身不是刺激，「接近力竭」才是觸發訊號
- 留 RIR 1–3（還剩 1–3 下才力竭）→ 足夠刺激 ＋ 恢復更快 ＋ 保留應變餘裕

### 抑制迴路強化：重複激活，不靠損傷

| | 肌肉 | 前額葉抑制迴路（PFC） |
|--|------|---------------------|
| 強化機制 | 微撕裂 → 修復 → 超補償 | 赫布定律：重複激活 → LTP 突觸增強 |
| 疲勞形式 | 收縮蛋白損耗 ＋ 代謝廢物 | 神經傳導物質（多巴胺）局部耗盡 |
| 恢復時間 | 24–48 小時 | 數小時，慢性消耗則更長 |
| 過度使用後果 | 過度訓練、受傷 | PFC 長期耗竭、決策品質下降 |
| 最適訓練方式 | RIR 1–3，不每次榨乾 | 適量練習，留恢復餘裕 |

共同原則：強化發生在恢復期，不是在練習當下；「不把自己累死」在身體和認知層面都成立。

### CEN vs TPN（腦網路澄清）

- **TPN（任務正向網路）**：傘狀概念，所有執行任務時啟動的網路
- **CEN（中央執行網路）**：TPN 的子集，dlPFC ＋ 後頂葉，負責工作記憶與認知控制
- **SN（顯著性網路）**：TPN 的另一子集，前扣帶迴 ＋ 腦島，負責偵測干擾訊號並切換 DMN ↔ CEN
- **DMN（預設模式網路）**：TPN 的對立面，發呆/渴望/白日夢時啟動

## 情境區（尚未分層）

### 情境一：外部要求觸發的動機崩解鏈

**觸發點**：老師要求交筆記（可泛化至任何他人強加的要求）

**完整鏈條**：

```
外部要求出現
     ↓
動機從「認同型」切換成「外在型」（SDT）
「我在為他做，不是為我做」
     ↓
皮質醇上升（被迫做不認同的事 = 慢性壓力）
多巴胺 D_base 下降（自主感消失）
     ↓
認知到「這對長期發展沒幫助」→ 認知失調
     ↓
進入「騙自己階段」（內攝調節，Introjected Regulation）
「為了成績還是要做」
     ↓
知道自己在騙自己 → 失調加深，PFC 資源持續消耗
     ↓
壓制系統承載不住
     ↓
情緒爆發 → 行為能力喪失
```

**關鍵洞見**：
- 「滿足要求」和「滿足自己」是兩種動機結構，生化結果截然不同
- 自主達成 → 睪固酮小幅上升 ＋ D_base 補充 ＋ 皮質醇下降
- 應付要求 → 睪固酮不上升 ＋ D_base 無補充 ＋ 皮質醇持平或上升
- 被動性自我欺騙比外在壓力更消耗，因為多了「我知道我在騙自己」的認知成本

**三層抗拒的機制對應**：
- 抗拒一（怕被解讀成裝認真）→ 社交評判型皮質醇（L2 皮質醇觸發情境三）
- 抗拒二（只是評分，無長期幫助）→ L3 維度三：完全在被動端
- 抗拒三（細分後顯得麻煩）→ PFC 對未知微決策數量的超估，和任務本身難度無關

**自主目標重構原則**：
把「完成他人要求」換成「完成一件對自己有意義的事，交出去是副產品」。
抗拒三的處理：只定義第一步（如「打開資料夾」），不讓 PFC 預估整條路的決策成本。

**潛在的 L 層歸屬方向**（待決定）：
可能屬於「皮質醇觸發情境」（L2 延伸）或獨立的行為動態層。

## 受體敏感度——激素的音量旋鈕

激素水平是訊號強度，受體敏感度決定訊號能被接收多少。

| 受體 | 提升敏感度 | 降低敏感度 |
|------|-----------|-----------|
| 雄激素受體（AR）← 睪固酮 | 阻力訓練、充足睡眠 | 慢性皮質醇、睡眠不足 |
| 多巴胺 D2 受體 | 有氧與阻力運動、低刺激期 | 慢性高刺激（遊戲、社群媒體）→ 耐受性 |
| 糖皮質激素受體（GR）← 皮質醇 | — | 慢性皮質醇 → 下調 → 炎症調節失效 |

**D2 受體下調的惡性循環**：遊戲 → D2 下調 → 日常活動更無聊 → 需要更多刺激 → 更多遊戲。D2 恢復需要 2–4 週低刺激期。

## 性高潮對激素與 MPS 的影響

**高潮時序**：多巴胺急衝（自然界最高峰值之一）→ 催產素上升（降皮質醇）→ 高潮後泌乳素顯著上升 1–2 小時 → 睪固酮微跌後回基線。

**是否影響肌肉量？不足以**。
- 泌乳素升高與睪固酮微跌：小時級別
- MPS 有效窗口：天級別
- 兩者時間尺度不匹配，無法在 MPS 層面競爭

**長期淨評估（以 MPS 為目標）**：
- 健康性生活 → 催產素長期降皮質醇 → AR 敏感度維持 → 淨正向
- 7 天禁慾的睪固酮小幅峰值（Jiang 2003）：短暫且微小，無法抵消慢性皮質醇損害
- 訓練前 1–2 小時內高潮：泌乳素仍高，次優；訓練後高潮無影響

**高潮後的 D_peak 崩落**是自然界最深的多巴胺崩落之一（泌乳素介導的強制 D_reset）。

## 不確定性稅——迴避反應的根本來源與自我應驗機制

**三層抗拒皆為不確定性的不同形式**：
- 社交壓力（「別人怎麼解讀？」）→ 社交結果不確定性
- 缺乏長期動機（「對長期有沒有幫助？」）→ 未來價值不確定性
- 任務複雜度（「細分後有多少步驟？」）→ 執行路徑不確定性

**放大機制**：不確定性稅（疊加皮質醇）→ PFC 進入威脅敏感模式 → 成本估算系統被放大 → 細分時重估落差更大 → 迴避反應更強。

不是任務真的更難，是估算基準被皮質醇拉高了。

**自我應驗循環**：
迴避 → 任務停在「未知」狀態 → 不確定性從未解決 → 下次更快觸發不確定性稅 → 「果然很麻煩」得到驗證 → 迴避模式被學習強化。

**打破方法**：只定義第一步（限制模擬地平線），讓 PFC 只需估算一個動作的成本，使重估落差歸零。

## 勝利效應（Winner Effect）與小目標的乘數機制

- 達成**自己設立**的目標 → 睪固酮短暫上升（勝利效應，非體力競爭也成立）
- 睪固酮上升 → 抑制 UPS、提升 MPS 效率
- 同樣的訓練量，在好的激素環境下產生更多肌肉
- 小目標是**乘數**，不是替代訓練刺激的手段
- 自主感是觸發勝利效應的必要條件——他人強加的目標達成後效應較弱或不存在

## MPS vs MPB 九條有效路徑

**MPS**（Muscle Protein Synthesis，肌肉蛋白質合成率）提高路徑：

| # | 路徑 | 機制 |
|---|------|------|
| 1 | 訓練（機械張力）| 接近力竭 → mTOR 活化 → MPS↑ |
| 2 | 蛋白質攝取 | 白胺酸門檻（20–25g 一次）→ mTORC1 → MPS↑ |
| 3 | 睪固酮↑ | 低皮質醇 → 孕烯醇酮流向睪固酮 → 雄激素受體活化 → MPS↑ |
| 4 | 深眠 | GH 釋放 → IGF-1 → MPS↑ |
| 5 | 勝利效應 | 自主目標達成 → 睪固酮短暫升高 → MPS 效率提升 |

**MPB**（Muscle Protein Breakdown，肌肉蛋白質分解率）降低路徑：

| # | 路徑 | 機制 |
|---|------|------|
| 6 | 皮質醇↓ | 低壓力／催產素↑ → UPS（MuRF1/MAFbx）活性↓ → MPB↓ |
| 7 | 能源充足 | 蛋白質＋碳水足夠 → 不啟動糖質新生 → 肌肉不被當燃料分解 |
| 8 | MED 訓練 | 不過度訓練 → 皮質醇不慢性偏高 → MPB 不持續被刺激 |
| 9 | 睡眠 | 夜間皮質醇維持低點 → MPB 基線低 |

核心洞見：「享受生活＋低壓力」一句話同時覆蓋 MPB 全部 4 條路徑，並透過睪固酮讓 MPS 更有效率。

## 訓練動作設計原則（今日確立）

### 力量—速度曲線與三種適應

```
力量大 ●  重量大＋速度慢 → 最大肌力（神經適應）
       ●
       ● ← 爆發力峰值（中等重量＋最快速度）
       ●
力量小 ●  重量輕＋速度快 → 速度∕爆發力
       ────────────────────
       慢                 快   速度
```

| 訓練類型 | 次數 | 特徵 | 主要適應 |
|---------|------|------|---------|
| 最大肌力 | 1–5 | 極重、接近力竭 | 神經效率、收縮蛋白密度 |
| 肌肥大 | 6–15 | 中重、接近力竭 | 肌肉截面積增大（視覺壯碩）|
| 爆發力 | 任意 | 輕∕中、最快速度 | 快縮肌（IIx）、RFD |
| 肌耐力 | 15–30＋ | 輕、高次數 | 微血管、粒線體、Type I |

### 最佳複合動作模式：慢離心＋快向心

- **蹲下（離心）**：2–3 秒慢速控制 → 機械張力最大、肌肥大主要來源
- **最低點**：短暫停頓加強張力
- **站起（向心）**：爆發式快速 → 招募快縮肌（IIx）、訓練爆發力
- **高次數做到力竭**：同時獲得微血管增生與肌耐力

此模式單一動作可同時訓練肌肥大＋爆發力＋肌耐力，三者不互斥。

### 微血管增生（Angiogenesis）機制

持續高次數使用肌肉 → 局部低氧 → VEGF 分泌 → 新生微血管。
微血管密度越高 → 養分與氧氣運輸越快 → 長期 MPS 環境更好。
高次數訓練比低次數更能觸發此適應。

### 蛋白質中毒（Rabbit Starvation）

只吃極瘦肉類（如野兔）會死亡的原因：
- 肝臟尿素合成速率上限約 200–300 g 蛋白質/天，超過後血氨累積中毒
- 蛋白質熱效應（TEF）高達 25–30%，大量熱量流失於消化過程
- 缺乏脂肪 → 膽固醇不足 → 睪固酮與細胞膜合成崩潰
- 安全蛋白質攝取上限：每日熱量的 35%（75 kg、2500 kcal ≈ 218 g）

### MPS 與脂肪合成的前驅物競爭

兩條路徑使用不同原料，不互相競爭：
- MPS：胺基酸（來自蛋白質）
- 脂肪合成：乙醯輔酶 A（來自碳水化合物與脂肪）
- 決定肌肉∕脂肪比例的是**熱量盈餘大小**，不是前驅物競爭
- 微量盈餘（+200–300 kcal）可優先導向 MPS，減少囤脂
