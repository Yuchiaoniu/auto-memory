## Context

使用者為大三下資管生，推甄目標為政大/台師大/交大資管所（2026/10–12月）。AWS CLF-C02 作為備審技術佐證，需在 2026/06 底前取得。備考資源以 Claude 教學為主，AWS 官方範例題為輔，無付費課程預算。

## Goals / Non-Goals

**Goals:**
- 在 7 週內系統性完成四大考試範圍
- 透過對話式學習掌握所有考試知識點
- 以週次任務管控進度，確保按時考試

**Non-Goals:**
- 不深入研究 AWS 實際操作（CLI / Console 操作非考試重點）
- 不購買付費練習題平台（Whizlabs / Udemy practice exam）
- 不追求 900+ 高分，目標為通過（720/1000）

## Decisions

**D1：以 Claude 對話為主要學習方式**
理由：使用者已有 Claude Code 環境，無需額外訂閱成本。對話式教學可針對弱點即時補強，比影片課程更有彈性。

**D2：週次任務拆分，對應四大範圍**
理由：CLF-C02 考試比重明確，按比重分配學習時間可最大化備考效率。Security（30%）＋Technology（34%）合計 64%，優先確保這兩塊扎實。

**D3：第三週集中模擬題，最後一週弱點補強**
理由：先建立知識基礎再做題，避免還沒學完就被錯題打擊信心。官方 20 題範例在第三週做，感受真實題型難度。

## Risks / Trade-offs

- [風險] 無付費練習題，題庫廣度不足 → 緩解：官方範例題 + Claude 針對性出題補充
- [風險] 七週時間偏緊（含考試報名等待） → 緩解：第五週起即可報名，不必等到第七週才排考
- [風險] 英文題目理解障礙 → 緩解：學習階段中文理解，考前做英文版範例題適應語感
