## Why

AWS CLF-C02 是推甄備審的技術佐證證照，計畫於 2026/06 底前考取。透過系統化備考流程，在六週內完成所有知識範圍並通過考試。

## What Changes

- 建立四大考試範圍的學習計畫與進度追蹤
- 以週為單位拆分備考任務，確保 2026/06/30 前完成考試
- 透過 Claude 教學 + AWS 官方範例題雙軌進行

## Capabilities

### New Capabilities

- `cloud-concepts`: Cloud 定義、優勢、架構設計原則、6R 遷移策略、Well-Architected Framework（佔比 24%）
- `security-compliance`: IAM、共同責任模型、加密、合規、Shield/WAF（佔比 30%）
- `cloud-technology-services`: 核心服務 EC2、S3、RDS、Lambda、VPC、CloudFront、Route53、SNS、SQS、CloudWatch（佔比 34%）
- `billing-pricing-support`: 費用模型（On-Demand/Reserved/Spot）、Pricing Calculator、Cost Explorer、支援方案（佔比 12%）

### Modified Capabilities

（無）
