## ADDED Requirements

### Requirement: Z 表互動查詢工具

系統 SHALL 提供互動式 Z 表查詢工具，使用者輸入任意 z 值（範圍 [-4, 4]），即時顯示對應的標準常態機率與視覺化陰影，並提供四種機率類型切換（半邊、左尾、右尾、雙側中段）。

#### Scenario: 輸入 z 即時更新

- **WHEN** 使用者在 z 輸入框輸入 1.96
- **THEN** 系統即時顯示 $P(0 \le Z \le 1.96) = 0.4750$，並在 PDF 曲線上以陰影標出 [0, 1.96] 區域

#### Scenario: 切換機率類型

- **WHEN** 使用者選擇「雙側中段」並輸入 z = 1.96
- **THEN** 系統回傳 $P(-1.96 \le Z \le 1.96) = 0.9500$，陰影區域改為 [-1.96, 1.96]

#### Scenario: 邊界值處理

- **WHEN** 使用者輸入 z = 0
- **THEN** 系統顯示 $P(0 \le Z \le 0) = 0.0000$ 並無陰影區域

### Requirement: t 表互動查詢工具

系統 SHALL 提供互動式 t 分配臨界值查詢工具，使用者輸入自由度 ν 與右尾面積 α，回傳臨界值 $t_{\nu, \alpha}$，並與對應 Z 值對比。

#### Scenario: 直接查臨界值

- **WHEN** 使用者輸入 ν = 10、α = 0.05
- **THEN** 系統回傳 $t_{10, 0.05} = 1.8125$，並顯示對照的 $Z_{0.05} = 1.645$

#### Scenario: 雙尾檢定快速切換

- **WHEN** 使用者選「雙尾 95%」（α = 0.025）、ν = 19
- **THEN** 系統回傳 $t_{19, 0.025} = 2.0930$ 與信賴區間範本說明

#### Scenario: 自由度未列出時的處理

- **WHEN** 使用者輸入 ν = 35（不在 hardcoded 表中）
- **THEN** 系統用線性插補（在 ν=30 與 ν=40 之間）計算結果並標註「插補值」

### Requirement: 精確計算（無紙本表精度限制）

Z 表工具 SHALL 使用 Abramowitz-Stegun 近似的 erf 函數即時計算，提供至少 4 位小數精度，且允許輸入任意小數值（不限於紙本表的 0.01 步進）。

#### Scenario: 非整數步進的 z 值

- **WHEN** 使用者輸入 z = 1.234（紙本表不存在的步進）
- **THEN** 系統仍能算出 P(0 ≤ Z ≤ 1.234) ≈ 0.3914 並顯示陰影

### Requirement: 共用樣式與 UX 一致性

兩個查表工具 SHALL 沿用現有 `.viz-btn`、`.symbol-legend` 等樣式，並使用既有 Canvas 繪圖規格（暖橘 PDF、金色標示），與其他互動元件視覺一致。

#### Scenario: 視覺一致性

- **WHEN** 使用者從 area-viz 切到 ztable-lookup
- **THEN** 介面元件樣式（按鈕、輸入框、Canvas 配色）保持一致
