## Context

桌面路徑為 `C:\Users\yuchi\OneDrive\Desktop`，目前已完成捷徑整理與暫存檔清理。剩餘工作為將散落檔案依類型分類移入對應資料夾。

## Goals / Non-Goals

**Goals:**
- 將安裝檔與壓縮包移至「安裝檔」資料夾
- 將圖片與影片移至「媒體」資料夾
- 將學校文件移至適當資料夾

**Non-Goals:**
- 不整理現有資料夾內部的檔案
- 不刪除任何非暫存的使用者檔案
- 不修改任何程式或系統設定

## Decisions

- 每個步驟執行前先列出受影響檔案，讓使用者確認後再移動
- 目標資料夾若不存在則自動建立
- 使用 PowerShell 的 `Move-Item` 移動檔案

## Risks / Trade-offs

- [風險] 移動後找不到檔案 → 每步驟前預覽清單，使用者確認後才執行
- [風險] 部分檔案可能正在使用中 → 若移動失敗會顯示錯誤訊息，不強制執行
