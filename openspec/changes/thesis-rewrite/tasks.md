### 關鍵機制（值得寫入論文）
- **clusterByRelativeDiff 演算法**（lines 256-276）：以 10% 相對差為閾值的貪心聚類，
  解決 Path 0 跨幀中位數混合不同實際量測值之問題（取最大群聚之中位數）。
- **referenceAtTrunk 隱式 QC**：要求參照物實際貼附樹幹上，避免空間距離不同導致比例失真。
- **graceful degradation 設計原則**：高品質輸入（量尺貼於樹幹）→ Path 0 高精度；
  中等輸入（參照物在場）→ Path A 中精度；最低輸入 → Path B 仍可產出可用結果。
- **leafFrameIndices**：以幀索引標註含葉影格，供 PlantNet/iNaturalist 樹種辨識使用。
- **trunkWidthFraction > 0.8 強制拒絕**：避免樹幹超出畫面導致比例換算失真。
