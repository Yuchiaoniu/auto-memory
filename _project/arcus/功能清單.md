# Arcus 功能清單

> 最後更新：2026-07-13
> 架構原則（走甲）：不 import 跨檔、不開子行程，整個引擎合併成兩支 `.py`。
> - `server.py`：Flask HTTP 服務，port 7800，對外路徑前綴 `/arcus/`。
> - `arcus_core.py`：後端核心（四支合體＋跨檔基礎設施），server.py 具名匯入其中函式。
> - `workflow_gate_hook.py` 屬於另一層：Claude Code 的 PreToolUse 工作流閘門，
>   由 `~/.claude/settings.json` 掛鉤呼叫，不是 arcus 引擎的一部分。

## 現行功能群（arcus_core.py 內實際存在）

| 群 | 原始來源 | 內容 | 狀態 |
|---|---|---|---|
| 第一群 中央 log | write_log.py | JSONL 原子追加、永不拋例外（write_log／read_log／append_log_turn 等） | 生效 |
| 第二群 token 估算 | context_meter.py | est_tokens：中日韓字元 ×0.6、其餘 ÷4 | 生效 |
| 第三群 判官 | smart_brake.py | turn_review／build_turn_review：每輪工作明細，到期時 opus 判官複核 | 生效（判官開；斷點煞車機制已不存在） |
| 第四群 伺服器掛鉤 | arcus_hooks.py | before_prompt_hook（注入結構地圖＋函式索引）、after_response_hook（現為無副作用空操作） | 生效 |

### 跨檔基礎設施（併入 arcus_core.py，非上列四支之一）
- 結構地圖：`generate_project_map`（os.walk 掃專案、7 天內異動加 ★、雜湊快取）。
- 紀錄檔封存：`archive_token_log_jsonl_if_needed`（token log 逾兩天自動封存）。
- 函式索引：`build_symbol_index`（掃 server.py＋arcus_core.py 產生帶行號索引，每輪注入穩定前綴）。
- 記錄檔↔寫入函式對照 `_RECORD_WRITER_MAP`：告訴執行中的 agent「哪個記錄檔由哪個函式寫」，
  避免重跑 grep。已於 2026-07-13 清掉指向不存在函式的 5 條死行。

### server.py 具名匯入的核心函式（17 個，皆存在）
before_prompt_hook、after_response_hook、generate_project_map、read_utf8、
read_log_for_display、append_log_turn、trim_log_if_needed、_tool_summary、
append_token_log、_strip_map_footer、build_system_prompt、parse_image_with_claude、
build_turn_review、build_tasks_injection、est_tokens、write_log、read_log。

## 已移除功能群（2026-07-13）

| 原群 | 原始來源 | 原宣稱內容 | 為何移除 |
|---|---|---|---|
| 反向傳播基因反思 | weight_baker.py | `bake()`：依 fall_counter 反向傳播、只改 DNA_weights.json 數字 | 先前版本已被掏空成「註解標頭＋大段空白行」，本檔無任何實作、server.py 也未匯入 |
| 法條層＋事後偵測 | routing_law.py | `run()`／`scan()`：法條短路、違規記錄、軌跡向量、path-index | 同上，整段是空殼殘留 |

連帶清理：
- 斷點煞車（mark_turn_start／read_checkpoint／run_smart_brake）與死開關 `_SMART_BRAKE_ON`——
  函式本體早已不存在，開關無任何程式讀取，屬純殘留，一併刪除。
- `DNA_weights.json`：因 `bake()` 已不存在，本檔不再改寫它，現為靜態快照。

## 重要影響
標頭原本宣稱「六支合體」，但第三、四群是空殼；且 `_RECORD_WRITER_MAP` 與
`ARCUS_ARCHITECTURE` 每輪都注入系統提示，等於每回合對執行中的 agent 謊報它有
`bake()`／`scan()`／action_trace／court-cases 這些不存在的能力。清理後宣稱與實作一致。
